# Pair Class
This class is just a simple representation of a Pair of values.  
It is used in some of the other classes of this library internally and some other libraries that I am going to be importing to Star Development LLC at some point. I have found it quite useful. You just have to know which ones are which :).
I may add more classes for other amounts of parameters, `Triple`, `Quadruple`, `Septuple`... Name suggestions and a recommended amount of these are welcome.
```java
Pair<Integer, String> pairExample = new Pair<>(100, "One Hundred");
```
